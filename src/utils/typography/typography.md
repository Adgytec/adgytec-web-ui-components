# Typography Utility

The typography utility provides a set of constants representing the available typography variants in the design system. These constants map to CSS class names defined in the global styles.

## Constants

### `typography`

Standard typography variants based on Material 3 guidelines.

| Category | Variants |
|----------|----------|
| **Display** | `displayLarge`, `displayMedium`, `displaySmall` |
| **Headline** | `headlineLarge`, `headlineMedium`, `headlineSmall` |
| **Title** | `titleLarge`, `titleMedium`, `titleSmall` |
| **Body** | `bodyLarge`, `bodyMedium`, `bodySmall` |
| **Label** | `labelLarge`, `labelMedium`, `labelSmall` |

*Note: All variants also have an `Emphasized` version (e.g., `titleMediumEmphasized`).*

### `fluidTypography`

Fluid typography variants that scale smoothly based on the viewport size.

| Category | Variants |
|----------|----------|
| **Display** | `displayLarge`, `displayMedium`, `displaySmall` |
| **Headline** | `headlineLarge`, `headlineMedium`, `headlineSmall` |

## Usage

```tsx
import { typography, fluidTypography } from '@adgytec/web-ui-components';
import clsx from 'clsx';

function MyComponent() {
  return (
    <div className={clsx(typography.headlineLarge)}>
      This is a standard headline
    </div>
    
    <div className={clsx(fluidTypography.displayLarge)}>
      This is a fluid display text
    </div>
    
    <p className={clsx(typography.bodyMediumEmphasized)}>
      This is emphasized body text
    </p>
  );
}
```
