# Form Utility

The form utility provides helpers for extracting and validating data from HTML forms using [Zod](https://zod.dev/).

## `validateAndGetFormValues` Function

Extracts values from an `HTMLFormElement` using `FormData` and validates them against a Zod schema.

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `formElement` | `HTMLFormElement` | The form element to extract values from. |
| `schema` | `z.ZodTypeAny` | The Zod schema to validate the values against. |

### Returns

Returns a `FormValidateResult<T>` object:

**If validation succeeds:**
```ts
{
  success: true;
  data: T; // The validated and typed data
}
```

**If validation fails:**
```ts
{
  success: false;
  errors: Partial<Record<keyof T, string | string[]>>; // Field-specific error messages
}
```

## Usage

```tsx
import { validateAndGetFormValues } from '@adgytec/web-ui-components';
import * as z from 'zod';

const loginSchema = z.object({
  email: z.string().email(),
  password: z.string().min(8),
});

function LoginForm() {
  const handleSubmit = (event: React.FormEvent<HTMLFormElement>) => {
    event.preventDefault();
    
    const result = validateAndGetFormValues(event.currentTarget, loginSchema);
    
    if (result.success) {
      console.log('Form data:', result.data);
      // Perform login...
    } else {
      console.error('Validation errors:', result.errors);
      // Display errors to user...
    }
  };

  return (
    <form onSubmit={handleSubmit}>
      <input name="email" type="email" />
      <input name="password" type="password" />
      <button type="submit">Login</button>
    </form>
  );
}
```
