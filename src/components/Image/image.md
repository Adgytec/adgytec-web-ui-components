# Image

The `Image` component is a responsive image wrapper that simplifies the use of `srcset` and `sizes` attributes. It provides pre-defined presets for common image layouts and requires multiple source URLs for different resolutions.

## Props

The `Image` component extends the standard `img` element but replaces the `src` and `srcSet` props with specific resolution-based props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `thumbnail` | `string` | **Required** | URL for thumbnail resolution (150w). |
| `small` | `string` | **Required** | URL for small resolution (320w). Also used as the default `src`. |
| `medium` | `string` | **Required** | URL for medium resolution (640w). |
| `large` | `string` | **Required** | URL for large resolution (1280w). |
| `extraLarge` | `string` | **Required** | URL for extra large resolution (1920w). |
| `preset` | `Preset` | `"default"` | A layout preset that determines the `sizes` attribute. |
| `alt` | `string` | `""` | Alt text for the image. |
| `sizes` | `string` | — | Overrides the `sizes` attribute generated by the `preset`. |

### Presets

Presets define the `sizes` attribute to help the browser select the most appropriate image source.

| Preset | Generated `sizes` Attribute |
|--------|----------------------------|
| `default` | `(max-width: 50rem) 100vw, 50rem` |
| `thumbnail` | `10rem` |
| `normal` | `(max-width: 60rem) 100vw, 60rem` |
| `wide` | `(max-width: 80rem) 100vw, 80rem` |
| `full` | `100vw` |
| `grid3Col` | `(max-width: 40rem) 100vw, (max-width: 80rem) 50vw, 30rem` |

## Usage

```tsx
import { Image } from '@adgytec/web-ui-components';

<Image
  alt="A beautiful sunset"
  preset="wide"
  thumbnail="/images/sunset-150.jpg"
  small="/images/sunset-320.jpg"
  medium="/images/sunset-640.jpg"
  large="/images/sunset-1280.jpg"
  extraLarge="/images/sunset-1920.jpg"
/>
```
