# Toggle Button

## Configurable CSS Tokens

| Token | Description |
|-------|-------------|
| `--md-toggle-button-background` | Background color of the button in its unselected state. |
| `--md-toggle-button-selected-background` | Background color of the button in its selected state. |
| `--md-toggle-button-color` | Text and icon color in the unselected state. |
| `--md-toggle-button-selected-color` | Text and icon color in the selected state. |
| `--md-toggle-button-outline-color` | Outline color in the unselected state (for `outlined` color variant). |
| `--md-toggle-button-selected-outline-color` | Outline color in the selected state. |

The `ToggleButton` component allows users to switch between two states (e.g., On/Off, Active/Inactive) with a button interface. It implements Material 3 toggle button patterns.

## Props

The `ToggleButton` component extends [React Aria Components ToggleButton](https://react-spectrum.adobe.com/react-aria/ToggleButton.html).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `"filled" \| "tonal" \| "outlined" \| "elevated"` | `"filled"` | The visual style and color scheme. |
| `size` | `"extra-small" \| "small" \| "medium" \| "large" \| "extra-large"` | `"medium"` | The size of the button. |
| `shape` | `"square" \| "rounded" \| "full"` | `"full"` | The corner radius of the button. |
| `icon` | `LucideIcon` | — | An optional icon displayed within the button. |
| `selectedIcon` | `LucideIcon` | — | An optional icon to display when the button is selected. If not provided, `icon` is used for both states. |
| `iconPlacement` | `"start" \| "end"` | `"start"` | The position of the icon relative to the text. |
| `tooltip` | `string` | — | Optional tooltip text displayed on hover. |

## Usage

```tsx
import { ToggleButton } from '@adgytec/web-ui-components';
import { Star, StarOff } from 'lucide-react';

<ToggleButton 
  icon={StarOff} 
  selectedIcon={Star}
  aria-label="Bookmark"
>
  Bookmark
</ToggleButton>

<ToggleButton color="outlined" isSelected={true}>
  Active
</ToggleButton>
```
