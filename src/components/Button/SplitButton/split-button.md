# Split Button

The `SplitButton` component allows users to take a primary action and choose from a set of related secondary actions via a dropdown menu. It is composed of a primary action button and a dropdown trigger button.

## Configurable CSS Tokens

The `SplitButton` uses the same core button tokens as the standard `Button` component for its background, text, and outline colors.

| Token | Description |
|-------|-------------|
| `--md-button-background` | Background color of the split button parts. |
| `--md-button-color` | Text and icon color. |
| `--md-button-outline-color` | Outline color (specifically for the `outlined` color variant). |

## Components

The `SplitButton` system is composed of:

- `SplitButton`: The main container (based on `Toolbar`) that provides context (size, color, state) to its children.
- `SplitButtonPrimary`: The main action button.
- `SplitButtonTrigger`: The button that opens the related actions menu.

## Props

### `SplitButton`
Extends `Toolbar` (excluding `orientation`, `children`, `className`, and `style`).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `"filled" \| "tonal" \| "outlined" \| "elevated"` | `"filled"` | The visual style and color scheme. |
| `size` | `"extra-small" \| "small" \| "medium" \| "large" \| "extra-large"` | `"small"` | The size of the component. |
| `isPending` | `boolean` | `false` | Global loading state for both buttons. |
| `isDisabled` | `boolean` | `false` | Global disabled state for both buttons. |

### `SplitButtonPrimary`
Extends [React Aria Components Button](https://react-spectrum.adobe.com/react-aria/Button.html).

| Prop | Type | Description |
|------|------|-------------|
| `icon` | `LucideIcon` | An optional icon to display. |
| `iconPlacement` | `"start" \| "end"` | The position of the icon relative to the text. |
| `tooltip` | `string` | Optional tooltip text. |

### `SplitButtonTrigger`
Extends [React Aria Components Button](https://react-spectrum.adobe.com/react-aria/Button.html). Usually used within a `MenuTrigger`.

| Prop | Type | Description |
|------|------|-------------|
| `tooltip` | `string` | Optional tooltip text. |

## Usage

```tsx
import { 
  SplitButton, 
  SplitButtonPrimary, 
  SplitButtonTrigger,
  MenuTrigger,
  MenuPopover,
  Menu,
  MenuItem
} from '@adgytec/web-ui-components';
import { Save } from 'lucide-react';

<SplitButton color="tonal">
  <SplitButtonPrimary icon={Save} onPress={() => console.log('Save')}>
    Save
  </SplitButtonPrimary>
  
  <MenuTrigger>
    <SplitButtonTrigger aria-label="More save options" />
    <MenuPopover>
      <Menu>
        <MenuItem label="Save and execute" />
        <MenuItem label="Save as draft" />
      </Menu>
    </MenuPopover>
  </MenuTrigger>
</SplitButton>
```
