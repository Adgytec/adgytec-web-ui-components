# Snackbar

## Configurable CSS Tokens

| Token | Default Value | Description |
|-------|---------------|-------------|
| `--md-snackbar-background` | `var(--md-sys-color-inverse-surface)` | The background color of the snackbar. |
| `--md-snackbar-color` | `var(--md-sys-color-inverse-on-surface)` | The text and icon color of the snackbar. |
| `--md-snackbar-action-color` | `var(--md-sys-color-inverse-primary)` | The color of the action button in the snackbar. |

The `Snackbar` component is an implementation of [Material 3 Snackbars](https://m3.material.io/components/snackbar/overview). It provides brief messages about app processes at the bottom of the screen.

## Components

### SnackbarRegion

The `SnackbarRegion` component should wrap the part of your application that needs to display snackbars. It provides the context for the snackbar queue and manages the rendering of active snackbars.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The content of your application. |
| `maxVisibleSnackbars` | `number` | `1` | The maximum number of snackbars visible at once. |

### useSnackbarQueue

A hook that returns the snackbar queue, allowing you to trigger snackbars from any child component of `SnackbarRegion`.

## Usage

```tsx
import { SnackbarRegion, useSnackbarQueue, Button } from '@adgytec/web-ui-components';

function MyComponent() {
  const snackbarQueue = useSnackbarQueue();

  const showSnackbar = () => {
    snackbarQueue.add({
      supportingText: 'Message sent successfully',
      action: <Button variant="text" onPress={() => console.log('Undo')}>Undo</Button>,
    }, { timeout: 5000 });
  };

  return (
    <Button onPress={showSnackbar}>Show Snackbar</Button>
  );
}

function App() {
  return (
    <SnackbarRegion>
      <MyComponent />
    </SnackbarRegion>
  );
}
```

### Snackbar Content Options

When adding a snackbar to the queue, you can provide the following content options:

| Option | Type | Description |
|--------|------|-------------|
| `supportingText` | `ReactNode` | The main message of the snackbar. |
| `action` | `ReactNode` | (Optional) An action button or element. |
| `hideCloseAction` | `boolean` | (Optional) Whether to hide the default close (X) button. |
