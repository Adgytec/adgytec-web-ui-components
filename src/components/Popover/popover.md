# Popover

The `Popover` component is a container used to display content on top of other content, often triggered by another element. It features built-in enter and exit animations that respond to its placement.

## Props

The `Popover` component extends the [React Aria Components Popover](https://react-spectrum.adobe.com/react-aria/Popover.html) and accepts all of its props.

## Usage

Popovers are typically used within a trigger component like `DialogTrigger`, `MenuTrigger`, or `TooltipTrigger`.

```tsx
import { Popover, Button, DialogTrigger, Dialog } from '@adgytec/web-ui-components';

<DialogTrigger>
  <Button label="Open Popover" />
  <Popover>
    <Dialog>
      <div style={{ padding: '16px' }}>
        This is the content inside the popover.
      </div>
    </Dialog>
  </Popover>
</DialogTrigger>
```
