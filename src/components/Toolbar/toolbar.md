The `Toolbar` component is a container for a set of related actions, such as buttons, toggle buttons, and dividers. It follows Material 3 design patterns for layout and styling.

## Props

The `Toolbar` component extends the [React Aria Components Toolbar](https://react-spectrum.adobe.com/react-aria/Toolbar.html) and accepts all of its props.

### Additional Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"docked" \| "floating"` | `"docked"` | The visual variant of the toolbar. `"floating"` adds a shadow. |
| `color` | `"standard" \| "vibrant"` | `"standard"` | The color scheme of the toolbar. |

## Usage

```tsx
import { Toolbar } from '@adgytec/web-ui-components';
import { IconButton } from '../Button';
import { Bold, Italic, Underline } from 'lucide-react';

<Toolbar aria-label="Text formatting">
  <IconButton icon={Bold} aria-label="Bold" />
  <IconButton icon={Italic} aria-label="Italic" />
  <IconButton icon={Underline} aria-label="Underline" />
</Toolbar>

<Toolbar aria-label="Vibrant Toolbar" color="vibrant">
  <IconButton icon={Bold} aria-label="Bold" />
  <IconButton icon={Italic} aria-label="Italic" />
</Toolbar>
```
