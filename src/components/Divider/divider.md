# Divider

## Configurable CSS Tokens

| Token | Default Value | Description |
|-------|---------------|-------------|
| `--md-divider-color` | `var(--md-sys-color-outline-variant)` | The color of the divider line. |

The `Divider` component is an implementation of [Material 3 Dividers](https://m3.material.io/components/divider/overview). It is a thin line that groups content in lists and layouts.

## Props

The `Divider` component extends the [React Aria Components Separator](https://react-spectrum.adobe.com/react-aria/Separator.html) and accepts all of its props.

## Usage

```tsx
import { Divider } from '@adgytec/web-ui-components';

// Horizontal divider (default)
<Divider />

// Vertical divider
<div style={{ display: 'flex', height: '100px', alignItems: 'center' }}>
  <span>Left</span>
  <Divider orientation="vertical" />
  <span>Right</span>
</div>
```

### Customizing Color

You can customize the divider color using the `--md-divider-color` CSS variable:

```css
.my-custom-container {
  --md-divider-color: var(--md-sys-color-primary);
}
```
