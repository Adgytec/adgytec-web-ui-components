# Action Dialog

The `ActionDialog` is a pre-configured `Dialog` component that simplifies the creation of common dialog patterns, including a heading, an optional icon, content body, and action buttons.

## Props

The `ActionDialog` component extends the `Dialog` component (excluding `className`).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `heading` | `string` | — | The title text displayed in the dialog header. |
| `icon` | `LucideIcon` | — | An optional icon displayed above the heading (centered) or beside it. |
| `actions` | `ReactElement[] \| ((renderProps) => ReactElement[])` | — | An array of action buttons (usually `Button` components) to be displayed at the bottom of the dialog. |
| `divider` | `ActionDialogDividerPlacement` | `"none"` | Placement of dividers within the dialog (`"none"`, `"all"`, `"after-heading"`, `"before-actions"`). |

## Usage

```tsx
import { 
  ActionDialog, 
  Modal, 
  ModalOverlay, 
  DialogTrigger, 
  Button 
} from '@adgytec/web-ui-components';
import { AlertTriangle } from 'lucide-react';

<DialogTrigger>
  <Button label="Delete Account" color="error" />
  <ModalOverlay>
    <Modal>
      <ActionDialog
        icon={AlertTriangle}
        heading="Confirm Deletion"
        actions={[
          <Button key="cancel" label="Cancel" color="text" />,
          <Button key="confirm" label="Delete" />
        ]}
        divider="before-actions"
      >
        Are you sure you want to delete your account? This action cannot be undone.
      </ActionDialog>
    </Modal>
  </ModalOverlay>
</DialogTrigger>
```

### Layout Behavior

- **Icon + Heading**: If an `icon` is provided, the header section becomes center-aligned.
- **Dividers**: Dividers can be added to separate the header, body, and action areas for improved visual hierarchy.
- **Scrollable Body**: The body content area is automatically scrollable if the content exceeds the available space.
