# Loader

The `Loader` component is a simple spinning circular indicator used to signal that a process is underway. It is built using the `LoaderCircle` icon from Lucide.

## Props

The `Loader` component extends the `Icon` component, but hides the `icon` and `className` props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number \| string` | — | The size of the loader icon. |
| `color` | `string` | — | The color of the loader icon. |

## Usage

```tsx
import { Loader } from '@adgytec/web-ui-components';

// Default loader
<Loader />

// Custom size and color
<Loader size={32} color="var(--md-sys-color-primary)" />
```

## Animation

The loader features a continuous 360-degree rotation animation defined in `loader.module.css`.
