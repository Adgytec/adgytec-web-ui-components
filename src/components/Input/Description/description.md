# Description

The `Description` component is a helper used to display auxiliary or help text for an input field. It automatically assigns itself to the `description` slot for integration with React Aria Components.

## Props

Extends [React Aria Components Text](https://react-spectrum.adobe.com/react-aria/Text.html).

## Usage

```tsx
import { Description } from '@adgytec/web-ui-components';

<Description>Helpful text for the user.</Description>
```
