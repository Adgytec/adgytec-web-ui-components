# Label

The `Label` component is a helper used to display the title or label for an input field. It automatically assigns itself to the `label` slot for integration with React Aria Components.

## Props

Extends [React Aria Components Label](https://react-spectrum.adobe.com/react-aria/Label.html).

## Usage

```tsx
import { Label } from '@adgytec/web-ui-components';

<Label>Email Address</Label>
```
