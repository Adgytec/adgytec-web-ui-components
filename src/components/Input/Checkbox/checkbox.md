# Checkbox

## Configurable CSS Tokens

| Token | Default Value | Description |
|-------|---------------|-------------|
| `--md-checkbox-outline-color` | `var(--md-sys-color-on-surface-variant)` | Border color of the checkbox when unselected. |
| `--md-checkbox-container-color` | `var(--md-sys-color-primary)` | Background color of the checkbox when selected. |
| `--md-checkbox-interaction-outline-color` | `var(--md-sys-color-on-surface)` | Border color when hovered, pressed, or focused. |
| `--md-checkbox-icon-color` | `var(--md-sys-color-on-primary)` | Color of the checkmark icon. |
| `--md-checkbox-state-layer-color` | `var(--md-sys-color-on-surface)` | Color of the state layer (ripple) in unselected state. |
| `--md-checkbox-selected-state-layer-color` | `var(--md-checkbox-container-color)` | Color of the state layer (ripple) in selected state. |

The `Checkbox` component is an implementation of [Material 3 Checkboxes](https://m3.material.io/components/checkbox/overview). It allows users to select one or more items from a set.

## Components

- `Checkbox`: An individual checkbox component.
- `CheckboxGroup`: A container for a group of checkboxes with support for label, description, and error messages.

## Props

### `Checkbox`
Extends [React Aria Components Checkbox](https://react-spectrum.adobe.com/react-aria/Checkbox.html).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `labelPlacement` | `"start" \| "end"` | `"end"` | Placement of the label relative to the checkbox indicator. |
| `containerStateLayer` | `boolean` | `false` | If true, the state layer (hover/focus) covers the entire checkbox container instead of just the indicator. |

### `CheckboxGroup`
Extends [React Aria Components CheckboxGroup](https://react-spectrum.adobe.com/react-aria/CheckboxGroup.html) (excluding `children`).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `ReactNode` | — | The label for the checkbox group. |
| `description` | `ReactNode` | — | Help text displayed below the group. |
| `errorMessage` | `ReactNode \| ((v: ValidationResult) => ReactNode)` | — | Error message displayed when validation fails. |
| `checkboxItemsGap` | `number` | `24` | Vertical gap between checkbox items in pixels. |
| `labelPlacement` | `"start" \| "end"` | `"end"` | Default label placement for all checkboxes in the group. |
| `containerStateLayer` | `boolean` | `false` | Default state layer behavior for all checkboxes in the group. |
| `showDescriptionOnInvalid` | `boolean` | `false` | Whether to keep showing the description when the group is invalid. |

## Usage

### Individual Checkbox

```tsx
import { Checkbox } from '@adgytec/web-ui-components';

<Checkbox>I agree to the terms</Checkbox>
<Checkbox labelPlacement="start">Accept cookies</Checkbox>
```

### Checkbox Group

```tsx
import { CheckboxGroup, Checkbox } from '@adgytec/web-ui-components';

<CheckboxGroup 
  label="Notifications" 
  description="Select how you want to be notified"
>
  <Checkbox value="email">Email</Checkbox>
  <Checkbox value="sms">SMS</Checkbox>
  <Checkbox value="push">Push Notifications</Checkbox>
</CheckboxGroup>
```
