# ComboBox

## Configurable CSS Tokens

| Token | Default Value | Description |
|-------|---------------|-------------|
| `--md-textfield-focused-outline-color` | `var(--md-sys-color-primary)` | Color of the outline when the combo box input is focused or open. |

The `ComboBox` component combines a text input with a listbox, allowing users to filter a list of options or enter a custom value. It supports single and multiple selection.

## Components

- `ComboBox`: The main container.
- `ComboBoxTrigger`: The input and button combination that triggers the list.
- `ComboBoxPopover`: The popover containing the list.
- `SelectList`: The list of items (shared with `Select`).
- `SelectItem`: An individual option (shared with `Select`).

## Props

### `ComboBox`
Extends [React Aria Components ComboBox](https://react-spectrum.adobe.com/react-aria/ComboBox.html).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `ReactNode` | — | The label for the combo box. |
| `description` | `ReactNode` | — | Help text displayed below the field. |
| `errorMessage` | `ReactNode \| ((v: ValidationResult) => ReactNode)` | — | Error message displayed when validation fails. |
| `showDescriptionOnInvalid` | `boolean` | `false` | Whether to keep showing the description when invalid. |
| `hideMultiSelectionValue` | `boolean` | `false` | If true, selected tags are not displayed in the container (useful for custom layouts). |

## Usage

### Single Selection

```tsx
import { 
  ComboBox, 
  ComboBoxTrigger, 
  ComboBoxPopover, 
  SelectList, 
  SelectItem 
} from '@adgytec/web-ui-components';

<ComboBox label="User">
  <ComboBoxTrigger placeholder="Select a user" />
  <ComboBoxPopover>
    <SelectList>
      <SelectItem id="1" textValue="John Doe">John Doe</SelectItem>
      <SelectItem id="2" textValue="Jane Smith">Jane Smith</SelectItem>
    </SelectList>
  </ComboBoxPopover>
</ComboBox>
```

### Multiple Selection (Tags)

When `selectionMode="multiple"` is used, selected items are displayed as removable tags within the container.

```tsx
<ComboBox label="Tags" selectionMode="multiple">
  <ComboBoxTrigger placeholder="Add tags..." />
  <ComboBoxPopover>
    <SelectList>
      <SelectItem id="react" textValue="React">React</SelectItem>
      <SelectItem id="vite" textValue="Vite">Vite</SelectItem>
      <SelectItem id="aria" textValue="React Aria">React Aria</SelectItem>
    </SelectList>
  </ComboBoxPopover>
</ComboBox>
```
