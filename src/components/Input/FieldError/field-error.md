# Field Error

The `FieldError` component is a helper used to display validation error messages for an input field. It is only visible when the associated input is in an invalid state.

## Props

Extends [React Aria Components FieldError](https://react-spectrum.adobe.com/react-aria/FieldError.html).

## Usage

```tsx
import { FieldError } from '@adgytec/web-ui-components';

<FieldError>This field is required.</FieldError>
```
