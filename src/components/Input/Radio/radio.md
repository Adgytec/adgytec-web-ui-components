# Radio

## Configurable CSS Tokens

| Token | Default Value | Description |
|-------|---------------|-------------|
| `--md-radio-icon-color` | `var(--md-sys-color-on-surface-variant)` | Border color of the radio when unselected. |
| `--md-radio-selected-icon-color` | `var(--md-sys-color-primary)` | Color of the radio indicator and border when selected. |
| `--md-radio-interaction-icon-color` | `var(--md-sys-color-on-surface)` | Border color when hovered, pressed, or focused (unselected). |
| `--md-radio-selected-interaction-icon-color` | `var(--md-radio-selected-icon-color)` | Border color when hovered, pressed, or focused (selected). |
| `--md-radio-state-layer-color` | `var(--md-sys-color-on-surface)` | Color of the state layer (ripple) in unselected state. |
| `--md-radio-selected-state-layer-color` | `var(--md-radio-selected-icon-color)` | Color of the state layer (ripple) in selected state. |

The `Radio` component is an implementation of [Material 3 Radio Buttons](https://m3.material.io/components/radio-button/overview). It allows users to select a single option from a set.

## Components

- `Radio`: An individual radio button component.
- `RadioGroup`: A container for a set of radio buttons with support for label, description, and error messages.

## Props

### `Radio`
Extends [React Aria Components Radio](https://react-spectrum.adobe.com/react-aria/RadioGroup.html#radio).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `labelPlacement` | `"start" \| "end"` | `"end"` | Placement of the label relative to the radio indicator. |
| `containerStateLayer` | `boolean` | `false` | If true, the state layer (hover/focus) covers the entire radio container instead of just the indicator. |

### `RadioGroup`
Extends [React Aria Components RadioGroup](https://react-spectrum.adobe.com/react-aria/RadioGroup.html) (excluding `children`).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `ReactNode` | — | The label for the radio group. |
| `description` | `ReactNode` | — | Help text displayed below the group. |
| `errorMessage` | `ReactNode \| ((v: ValidationResult) => ReactNode)` | — | Error message displayed when validation fails. |
| `radioItemsGap` | `number` | `24` | Gap between radio items in pixels. |
| `labelPlacement` | `"start" \| "end"` | `"end"` | Default label placement for all radios in the group. |
| `containerStateLayer` | `boolean` | `false` | Default state layer behavior for all radios in the group. |
| `showDescriptionOnInvalid` | `boolean` | `false` | Whether to keep showing the description when the group is invalid. |

## Usage

```tsx
import { RadioGroup, Radio } from '@adgytec/web-ui-components';

<RadioGroup 
  label="Storage Plan" 
  description="Select the plan that fits your needs"
  defaultValue="free"
>
  <Radio value="free">Free (5GB)</Radio>
  <Radio value="pro">Pro (100GB)</Radio>
  <Radio value="enterprise">Enterprise (Unlimited)</Radio>
</RadioGroup>
```

### Horizontal Layout

```tsx
<RadioGroup label="Select view" orientation="horizontal">
  <Radio value="list">List</Radio>
  <Radio value="grid">Grid</Radio>
</RadioGroup>
```
