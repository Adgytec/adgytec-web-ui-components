# Tabs

## Configurable CSS Tokens

| Token | Default Value | Description |
|-------|---------------|-------------|
| --md-tabs-color | `var(--md-sys-color-on-surface-variant)` | Text and icon color for inactive tabs. |
| `--md-tabs-active-color` | `var(--md-sys-color-on-surface)` | Text and icon color for the active tab. |
| `--md-tabs-active-indicator-color` | `var(--md-sys-color-primary)` | Color of the selection indicator line. |
| `--md-tab-list-divider-color` | `var(--md-sys-color-surface-variant)` | Color of the divider line below or beside the tab list. |

The `Tabs` component is an implementation of [Material 3 Tabs](https://m3.material.io/components/tabs/overview). It organizes content into high-level categories to help users find and move between sections of an application.

## Components

The `Tabs` system is composed of several components that extend [React Aria Components](https://react-spectrum.adobe.com/react-aria/Tabs.html):

- `Tabs`: The main container that manages selection state.
- `TabList`: A container for the individual `Tab` elements.
- `Tab`: An individual tab that users can interact with.
- `TabPanels`: A container for the content panels.
- `TabPanel`: The content associated with a specific tab.

## Props

### `Tab`
The `Tab` component extends the React Aria `Tab` but replaces its `children` with specific layout props.

| Prop | Type | Description |
|------|------|-------------|
| `label` | `ReactNode` | The text label to display. |
| `icon` | `LucideIcon` | An optional icon to display above or beside the label. |

### Other Components
`Tabs`, `TabList`, `TabPanels`, and `TabPanel` extend their respective React Aria Components props.

## Usage

```tsx
import { 
  Tabs, 
  TabList, 
  Tab, 
  TabPanels, 
  TabPanel 
} from '@adgytec/web-ui-components';
import { Home, Settings, User } from 'lucide-react';

<Tabs>
  <TabList aria-label="Account Settings">
    <Tab id="home" label="Home" icon={Home} />
    <Tab id="profile" label="Profile" icon={User} />
    <Tab id="settings" label="Settings" icon={Settings} />
  </TabList>
  <TabPanels>
    <TabPanel id="home">Home content goes here.</TabPanel>
    <TabPanel id="profile">Profile content goes here.</TabPanel>
    <TabPanel id="settings">Settings content goes here.</TabPanel>
  </TabPanels>
</Tabs>
```

### Vertical Orientation

```tsx
<Tabs orientation="vertical">
  <TabList aria-label="Vertical Tabs">
    <Tab id="v1" label="Tab 1" />
    <Tab id="v2" label="Tab 2" />
  </TabList>
  <TabPanels>
    <TabPanel id="v1">Panel 1</TabPanel>
    <TabPanel id="v2">Panel 2</TabPanel>
  </TabPanels>
</Tabs>
```
